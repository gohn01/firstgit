Configuring Git Discussion

  1. Go to VS code and click terminal --> New Terminal
  2. Check if git is already installed by typing git --version in the command line
  3. Configure git in your local computer
    • Set your username:
      git config --global user.name "your name"
    • Set your email address :
      git config --global user.email "your email address"
  4. To initialize the project directory type git init

Git add and coomit

  1. To see the status of the file in you directory type git status
  2. to track new file use the command git add:
    Syntax:
    git add .(to add groupu of file)
    git add filename(for individual file)
  3. By running the git status again you can see the stage files in green by tyoing
    Syntax:
    git commit -m "message for the developers"

Git Branches


Git branch is new version of the repository. It allows you to work on different part of the project without intacting the main branch

  1. To list branch/es type: git branch
  2. To create new branch git branch new-name
  3. To switch on a new branch git checkout branch-name

Create hotfixes Branches

TO create a new branches to deal with hotfixes without messing the with master branch directly type git checkout -b new-branch-name
